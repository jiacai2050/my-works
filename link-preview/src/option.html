Rule format One website per-line, format hostname,key1=value1,key2=value2.
Valid config keys are:
  • status: whether enable link preview, could be on/off
  • encoding: used for response decode, default is utf-8
  • max-length: per-site tooltip max length
  • position: per-site tooltip position
